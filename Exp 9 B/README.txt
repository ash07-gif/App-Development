TodoSQLiteApp - Kotlin (Android)

Description:
A simple To-Do List app using SQLite for persistent storage.
- Add tasks using EditText and Button
- View tasks in RecyclerView
- Delete tasks from the database
- Data persists even after closing the app

How to run:
1. Open this folder in Android Studio.
2. Let Gradle sync.
3. Run the app on an emulator or device.
